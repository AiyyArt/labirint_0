## Dependencies
- PyGame
- Numpy
- Matplotlib

## How to run
   - Clone the project
   - Inside the **game.py**, assign a path to the **PNG file** in the PDF_LOAD_PATH variable
     (For example: `PDF_LOAD_PATH = "C:/labirint_0/data/gen_18_1.png"`)
   - In terminal run: `$ python game.py`
